# ❤️ HeartNote

> Write from what actually happened.

HeartNote is a human-centered writing tool powered by AI that transforms real-life moments into meaningful, structured, and deeply personal writing.

---

## 🌟 Overview

HeartNote focuses on helping users express thoughts and emotions clearly.  
It is designed for real human experiences — not generic AI output.  

Instead of producing random text, it creates **context-aware, emotionally aligned writing** based on real user input.

---

## 🚀 Features

- ✍️ Emotion-aware writing generation  
- ⚡ Real-time responses powered by AI  
- 🧠 Context-based output (not generic text)  
- 💾 User authentication (Signup / Login / Delete Account)  
- 🗂️ Persistent database (PostgreSQL - Cloud)  
- 📱 Progressive Web App (PWA support)  
- 🎯 Clean and minimal UI for better focus  

---

## 🛠️ Tech Stack

**Frontend**
- HTML, CSS, JavaScript

**Backend**
- Python (Django)

**Database**
- PostgreSQL (Cloud - Render)

**Other**
- REST APIs  
- PWA (Service Workers, Manifest)

---

## 🧩 Architecture

User Input → Backend (Django) → AI Processing → Response Formatting → UI Display
